The navigation bar is an important element of a website's design since it allows users to quickly visit any section within the site. If you have ever visited a website without a navigation bar, you may have found it is difficult to locate the page you need. You may have also had to click "Back" several times in order to find a link to the next section you wanted to visit. Thankfully, web design has become more standardized in recent years and nearly every website now includes a navigation bar. What must you know for creating a navigation bar:

  1. You need to set ‹nav› element.
  2. You can choose 2 options for making a list inside in the ‹nav› element:
    1. You can use any element to open the dropdown content such as ‹span› or ‹button› element or
    2. You can make a list from the beginning to the end. It does not matter.
  3. Notice: If you make a horizontal navigation bar you must use float for your list.
  4. Do not forget about anchors, you have to set display: block in ‹a› element.
  5. Create first parent class where you will set position: relative which is positioned relative to its normal position.
  6. Create a second child class where you will set display: none and position: absolute which is positioned relative to the nearest positioned ancestor.
  7. Now you will write a rule about how those elements must work together. You will use a pseudo-class:hover for parent and to use a combinator for relationship between the selectors
  8. You the navigation bar is ready for use. Great job!
